Pytania dotyczące stylu lokalnego, liniowego, linijkowego

  1. Na co pozwala styl lokalny, który element będzie formatowany z użyciem stylu lokalnego

    2. Styl lokalny to sposób formatowania tekstu, który jest stosowany jedynie w określonym obszarze dokumentu, takim jak paragraf, nagłówek lub lista.


  2. Składnia teoretyczna
    3. Styl lokalny jest definiowany przy użyciu selektora, a następnie wprowadzając właściwości CSS i ich wartości. Na przykład:
    <style>
    [selektor] {cecha: wartość;}
    </style>

  3. Co może być selektorem?

    4. Selektory w CSS to wzorce, które określają, które elementy w strukturze dokumentu HTML zostaną sformatowane przez określone reguły stylów.

  4. Co wypisujemy jako "cechę". Jakie jest znaczenie tych elementów z użyciem operatoró w.

    5. Ceche w kontekście CSS odnosi się do właściwości, które chcemy zmienić lub dostosować w elemencie HTML

  5. Co wypisujemy jako :wartość"(w deklaracje stylu)?

    6. Jako "wartość" w deklaracji stylu wypisujemy wartość właściwości CSS, np. dla color wartość może być red, #FF0000, rgb(250, 0, 0)
