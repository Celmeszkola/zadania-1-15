✤ CSS wprowadza rozszerzone możliwości formatowania, takie jak cienie tekstów, gradienty czy transformacje, co umożliwia bardziej zaawansowane projektowanie stron internetowych.

✤ CSS oferuje elastyczne metody układania elementów za pomocą flexboxa i siatki (grid), co pozwala na tworzenie responsywnych i dynamicznych układów.

✤ Nowe możliwości CSS umożliwiają tworzenie interaktywnych efektów, takich jak zmiany podczas najechania kursorem (hover) czy przy aktywowaniu elementu (focus).

✤ CSS umożliwia tworzenie animacji za pomocą keyframes, co pozwala na dodanie płynności i dynamizmu do strony internetowej bez konieczności używania JavaScriptu.

✤ CSS oferuje rozszerzone możliwości kolorowania i tła, takie jak przezroczystość czy gradienty, co pozwala na tworzenie bardziej zaawansowanych efektów wizualnych.
